## How to use?

```
# Setup
mkdir -p ~/.local/bin/
npm config set prefix ~/.local
export PATH=~/.local/bin/:$PATH
npm install -g log4brains
```

```
log4brains adr new
```

Use `log4brains preview` to launch the site locally.
